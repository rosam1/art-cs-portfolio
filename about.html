about

This website demonstrates my work this summer on graphic and web design. I practiced applying the design process, learned how to manipulate basic elements of design, and how to build a custom website using HTML and CSS.

The design process involves: 1) identifying a problem - asking what the issue is that needs to be solved, who the design is for, and what are the logistics/constraints. Next, 2) research - how have other people addressed this design issue? Then 3) brainstorm - sketch or draw lots of ideas out for the next steps. 4) narrow down the ideas and 5) evaluate those ideas - does your design solve the original problem? how can it be improved? is all the content included? Finally, repeat 4) and 5) until you've come up with a good design.

In terms of graphic design, I learned how to create good design using principles such as balance, alignment, repetition, consistency, etc. I also got hands on experience creating images on paper and digitally, as well as using typography to add meaning and structure to a design. I also experimented with using shapes and color, and learned how to utilize different aspects of contrast (such as scale and direction) to create an engaging visual design and to guide the viewer's eye. Finally, I created two album covers for a fictional band that tied all of these different aspects of design together into one piece.

In terms of web design, I learned how to use HTML to put content on a webpage, and CSS to add style, with features such as layout, fonts and colors, transitions and animations, etc. I also learned how to use Bootstrap to add responsive JavaScript features like slideshows and popups. This website features a responsive layout for different screen sizes, a slideshow with linked thumbnails, a responsive navigation bar for the gallery page, enlarged images when you click on gallery images, and an accordion style folder in the about page.
